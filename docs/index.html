Overview

What This Project Does

A complete, production-ready ML system for intelligent resume screening — from raw CSV to deployment-ready model.

🎯

Objective

Predict whether a candidate will be Hired or Rejected based on resume data including skills, experience, education, certifications, job role, and salary expectations.

📊

Dataset

1,000 synthetic resume profiles with 11 columns. Target distribution: 812 Hires (81.2%) and 188 Rejects (18.8%). Certifications column has 274 missing values handled via imputation.

🧬

Feature Engineering

Custom multi-hot skill encoding, one-hot encoding for categoricals, StandardScaler for numerics, and a leakage-aware design that deliberately excludes the AI Score feature.

🤖

Model

Logistic Regression with saga solver and class_weight='balanced' for handling class imbalance. Wrapped in a full sklearn Pipeline for clean, reproducible inference.

⚠️ Leakage Check

Why AI Score Was Excluded

Including the AI Score (0-100) feature produced perfect 100% scores across all metrics — a clear sign of data leakage (the score likely encodes the hiring outcome directly).

The production model deliberately excludes this feature to ensure fairness and generalizability.

With AI Score 100% ← Leakage
Without AI Score 96% ← Fair Model

Quick Start 
# 1. Install dependencies
pip install -r requirements.txt

# 2. Run the pipeline
python src/ai_hiring_prediction_system.py \
    --data "data/AI-Based Hiring Prediction System (1) (1).csv"

# 3. (Optional) Include AI Score for leakage check
python src/ai_hiring_prediction_system.py \
    --data "data/AI-Based Hiring Prediction System (1) (1).csv" \
    --include-ai-score
